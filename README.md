# INEAPIpy

Este paquete de Python actua como Wrapper de la [API del INE](https://www.ine.es/dyngs/DAB/index.htm?cid=1099), permitiendo realizar peticiones utilizando una interfaz de Python.

## Instalación

```py
pip install ineapipy
```
